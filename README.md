### Bulgarian
- "ќ" only in fron of "o"
- "ѓ" only in fron of "o"

ще -> ште   
дзифт -> ѕифт   
джем -> џем   
кьопоолу -> ќопоолу   
гьол -> ѓол   
тайна -> тајна


### Macedonian
- "ќV" not "кјV"
- "ѓV" not "гјV"

љубов -> любов   
бања -> баня   
прст -> прст (stays the same)    
к’смет -> късмет

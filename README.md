# OneAlphabet

| IPA |  letter | IPA    | joted |
|-----|:-------:|:------:|----:
| a   | а       | ʲa/ja  | я |
| ɛ   | е       | ʲɛ/jɛ  | је |
| i   | и       | ʲi/i   | ји |
| ɔ   | о       | ʲɔ/jɔ  | јо|
| u	  | у       | ʲu/ju	 | ю |
| ɤ/ə | ъ       | ʲɤ/ʲə  | я*|



### Bulgarian
- "ќ" only in fron of "o"
- "ѓ" only in fron of "o"

ще -> ште   
дзифт -> ѕифт   
джем -> џем   
кьопоолу -> ќопоолу   
гьол -> ѓол   
тайна -> тајна


### Macedonian
- "ќV" not "кјV"
- "ѓV" not "гјV"

љубов -> любов   
бања -> баня   
прст -> прст (stays the same)    
к’смет -> късмет
